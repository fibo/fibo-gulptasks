fibo-gulptasks

common gulp tasks among projects

Installation

With npm do

$ npm install -g gulp
$ npm install gulp
$ npm install fibo/fibo-gulptasks

gulpfile

Then create a gulpfile.js

$ cat <<EOF > gulpfile.js
var gulp = require('gulp')
  , pkg  = require('./package.json')

require('fibo-gulptasks')(gulp, pkg)
EOF
