Installation

Install packages

npm install -g gulp
npm install --save-dev gulp fibo-gulptasks

Then create a gulpfile.js

cat <<EOF > gulpfile.js

var gulp = require('gulp')

require('fibo-gulptasks')(gulp)

EOF

Configuration

Config is stored in config.md file, you can see it in JSON format launching

gulp config

Development
